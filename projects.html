Pomodoro Study Timer

This project incorporated the concept of the Pomodoro effect. This study is in regard to how long a student can keep their attention span. The user starts the timer and it will count down for 25 minutes. Onnce the 25 minutes are complete, the user will be prompted to take a 5 minute break. This cycle will go on for 4 times before the 5 minute break turns into a 20 minute break. The techologies used here were Python, and the graphical user interface (GUI) used was Tkinter.

Kanye West API quote machine

The purpose of this project is to allow the user to click upon the emoji on the screen to produce a famous quote from Kanye West. Each time the emoji is clicked a new quote will appear. The technologies used in this project were Python, Tkinter, as well as using the requests module to retrieve the quotes via an API.
